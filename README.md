## Keylet -- Client library for Tillitis TKey

[![keylet on PyPI](https://img.shields.io/pypi/v/keylet.svg)](https://pypi.org/project/keylet/) [![keylet on GitHub](https://img.shields.io/badge/GitHub-jku%2Fkeylet-blue?logo=github)](https://github.com/jku/keylet) [![keylet documentation](https://img.shields.io/badge/Documentation-blue)](https://jku.github.io/keylet/)

keylet provides a client implementation for the [Tillitis TKey](https://www.tillitis.se/products/tkey/) security token, and specifically
ML-DSA signing using the [tkey-pq-device-signer](https://github.com/tillitis/tkey-pq-device-signer) device application.

See [documentation](https://jku.github.io/keylet/) for examples and API reference.
